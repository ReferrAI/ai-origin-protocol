# Contributing to ReferrAI / AIPP

Thanks for your interest in contributing! We aim to keep AIPP **minimal, privacy-safe, and interoperable**.

## Ways to Contribute
- Specs: propose clarifications to schemas or semantics
- Docs: improve diagrams, non-goals, and interoperability notes
- Implementations: add reference endpoints or test harnesses (optional)

## Pull Requests
1. Fork the repo and create a feature branch
2. Keep proposals small and scoped
3. Include examples where helpful
4. Submit a PR and reference any related issues
